## To generate APK file

1. Expo CLI installed globally

```bash
npm install -g expo-cli
```

2. Install EAS CLI

```bash
npm install -g eas-cli
```

3. Log in to Expo

```bash
eas login
```

4. Configure EAS Build

```bash
eas build:configure
```

5. Build the APK

```bash
eas build -p android --profile preview
```
